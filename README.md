## Deploy the system

### Client
```cd cliente```
And (
```php -S 0.0.0.0:8000```
Or
```python -m http.server 8000```
)

### Php Server
```
cd php/public
php -S 0.0.0.0:8001 php
```

### JavaScript Server
```
npm install
sudo npm i --save sqlite3 --unsafe-perm
cd nodejs
npm start
or
npm run dev
```
